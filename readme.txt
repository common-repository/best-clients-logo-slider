== Shortcode ==

(1) Use following shortcode to display clients, partners or sponsors Logo like Carousel logo slider.

[client_owl count="6" ]

(2)  Use following shortcode to display clients, partners or sponsors Logo like Logo Grid.
[client count="5" column="5" row="1" border="1px" p_id="41" ]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Dashboard > Clients to configure the plugin


== Frequently Asked Questions ==


= How do I add new Logo? =
To add new logo, go to Clients > Add New Client , then add Client Name, Client Logo , Client Email, Client Url. Then Publish it.

= What will be logo image dimension? =
Default image size set to 150px X 150 px.

= How to use shortcode in page templates? =
```<?php echo do_shortcode('[client_owl count="6" ]'); ?>```
